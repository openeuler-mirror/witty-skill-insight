# PR 113 说明（中文）

## 背景

本次更新解决了 `PR 113` 与 `PR 114` 的冲突，并将两边能力一起保留到 `dataset` 分支：

- 保留 `PR 113` 的数据集管理拆分与按场景组织能力
- 保留 `PR 114` 的关键动作自动抽取、控制流标注与执行轨迹解析能力

## 主要改动

### 1. 数据集管理保留按场景配置能力

- 数据集管理页继续区分 `Skill 召回率数据集` 与 `Skill 执行效果数据集`
- 执行效果数据集继续按场景聚合展示，便于按具体任务场景维护评测集
- 详情页保留场景级查看与复制能力

### 2. 执行效果数据集保留 PR 114 的关键动作能力

- 关键动作支持从 `Skill` 流程中自动抽取并复用，不再只依赖手工维护
- 页面会标明关键动作来源：`来自 Skill 流程自动抽取` / `来自共享关键动作配置`
- 页面会展示控制流信息，包括 `必选`、`条件分支`、`循环`、`可选`、`衔接`

### 3. Dashboard 与详情页同步增强

- Dashboard 中的执行效果数据集卡片会展示关键动作来源与控制流摘要
- 执行轨迹对比页保留自动解析与控制流展示
- Skill 详情页保留流程解析状态展示

### 4. 历史数据已完成回填

- 已为现有执行类数据集回填 `extractedKeyActions`
- 当前 public 数据中的执行类数据集已达到 `43 / 43` 全量补齐
- 其中大部分数据已具备控制流信息，可直接在页面上看到 `PR 114` 的展示效果
